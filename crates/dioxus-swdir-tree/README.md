# dioxus-swdir-tree

Directory-tree explorer widget for [Dioxus](https://dioxuslabs.com) GUI apps:
lazy loading, display filters, single & multi-selection, keyboard navigation,
drag & drop, prefetch, incremental search, and icon themes —
built on [`swdir`](https://crates.io/crates/swdir).

Provides `DirectoryTreeView` (the Dioxus component), `use_scan_driver` (the
async scan hook), and everything from `dioxus-swdir-tree-core` re-exported
for convenience.

```toml
[dependencies]
dioxus-swdir-tree = "0.7"
```

See the repository [README](https://github.com/nabbisen/dioxus-swdir-tree)
and [ROADMAP](https://github.com/nabbisen/dioxus-swdir-tree/blob/main/ROADMAP.md)
for full documentation and the feature list.
